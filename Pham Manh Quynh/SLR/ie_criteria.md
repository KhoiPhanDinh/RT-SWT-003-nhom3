# Inclusion and Exclusion Criteria

## Inclusion Criteria (IC)

Tất cả các tiêu chí dưới đây phải được thỏa mãn để paper được đưa vào nghiên cứu.

| ID  | Criteria                                                                                                                                                                 |
| --- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| IC1 | Paper is written in English.                                                                                                                                             |
| IC2 | Paper was published from 2024 onwards.                                                                                                                                   |
| IC3 | Paper is peer-reviewed (journal, conference, or workshop).                                                                                                               |
| IC4 | Paper focuses on mutation testing and applies AI/ML/LLM or intelligent optimization techniques (e.g., learning-based, search-based, evolutionary, AI-guided approaches). |
| IC5 | Paper includes an empirical evaluation with quantitative results (e.g., mutation score, mutant reduction, execution time, testing effort, cost reduction).               |

## Exclusion Criteria (EC)

A paper is excluded if it satisfies any of the following criteria.

| ID  | Criteria                                                                                                                   |
| --- | -------------------------------------------------------------------------------------------------------------------------- |
| EC1 | Duplicate paper.                                                                                                           |
| EC2 | Full-text PDF is not accessible.                                                                                           |
| EC3 | Poster, extended abstract, or paper shorter than 4 pages.                                                                  |
| EC4 | Not related to mutation testing (e.g., focuses only on fuzzing, test generation, test execution, or software maintenance). |
|     |                                                                                                                            |

## Screening Rule

A paper is included only if it satisfies **all Inclusion Criteria (IC1–IC5)** and does **not** satisfy any Exclusion Criteria (EC1–EC4).
