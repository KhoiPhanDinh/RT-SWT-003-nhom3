| Wang et al. | Tanhaei & Alizadehsani (2025), **GEM-LLM: Identifying contextual equivalent mutants via large language models; A global invariant-based approach**, Intelligent Systems with Applications | Dataset link in PDF: `https://github.com/tanhaei/GEM-LLM`; DOI/URL: **N/A** | GEM-LLM; GPT-4o v2024-05-13 and Gemma 3 27B; inter-procedural slicing + LLM invariant inference + SMT verification | 10 projects from Defects4J; Java mutants generated using PIT; focus on surviving mutants | Equivalent mutant detection rate; precision; computational overhead; operator-level detection rates | GEM-LLM classified **25%–30%** of overlooked surviving mutants as equivalent and achieved **98% precision**. | Borsi (2025), **Using Active Learning to Train Predictive Mutation Testing with Minimal Data**, ASE 2025 | DOI: `10.1109/ASE63991.2025.00217` | AL-PMT; active learning + random forest; skactiveml uncertainty sampling; sklearn random forest | Dataset from prior PMT work; base projects and GitHub projects; project-level experiments | F1 score; ROC-AUC; percentage of observed mutant kill status; training-data requirement; online/single-project performance | AL-PMT achieved **98%** of its best possible performance in over **80%** of examined projects while observing only **10%** of each project’s mutant-set kill status. Future work discusses continued predictions for future program versions and practical industry-oriented settings. |
